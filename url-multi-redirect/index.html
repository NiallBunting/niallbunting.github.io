Free URL Random Location Redirector

This page aims to create URLs that will direct to multiple pages. This is just done randomly. This means with one link a visitor could end up on different pages. This can be used for redirection for your users to multiple pages. Such as including a random button on your site.

You will probably want to put the URL in a shortener such as bit.ly.

Note: Please include the 'http(s)://' part of the url.
